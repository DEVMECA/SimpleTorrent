SimpleTorrent

https://play.google.com/store/apps/details?id=com.devmeca.simpletorrent

This application is a torrent application that will be built on top of Libre Torrent.

Additional function development, etc., will be developed in a different way from the existing Libre Torrent, and we will try to make the application more convenient through continuous updates.
